⚙️ Configuração do Firebase

Cole abaixo as credenciais do seu projeto Firebase. Elas ficam salvas apenas neste navegador e nunca são enviadas para nenhum servidor externo.

📖 Como obter essas credenciais?
  1. Acesse console.firebase.google.com e crie um projeto gratuito
  2. Clique em "Adicionar app" → Web e registre o app
  3. Copie cada valor do objeto firebaseConfig e cole nos campos acima
  4. No console Firebase: Firestore Database → Criar banco → Modo de teste
  5. Hospede no GitHub Pages e compartilhe o link com seu parceiro ✅
// Regras do Firestore (cole no console, aba "Regras"):
rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents/{doc=**} {
    allow read, write: if true;
  }
}
